# 使用单模型完成本地主线集成验收

重构后只训练一个 5-epoch 基础 Emo-FiLM 模型，并执行 `3→30→本地固定 2500 条`验证；其中 3 条是运行 smoke，30 条用于发现明显生成退化，2500 条用于本地固定数据合同上的 WER、Emo-SIM、DTW 和运行完整性统计。5 epoch、分阶段样本数和 `emofilm-eval-v2` 都是本地代理实验协议，不称作者源码或正式评测合同；结果只判断整体主线是否优于旧本地 final，不证明任一修复项的独立因果作用。旧 4×2 矩阵仅保留为历史定位证据，旧 1024d `per_emo_accuracy.py` 不进入最终验收。
